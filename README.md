# StreamByte 🚀

StreamByte is a premium, web-based Stremio client designed for a sleek, modern streaming experience. It integrates directly with Stremio addons to provide high-quality movies, TV shows, anime, and more.

## ✨ Features

- **Modern UI**: Dark-themed with purple accents, glassmorphism, and smooth Framer Motion animations.
- **Cross-Addon Search**: Search across all your integrated Stremio catalogs simultaneously.
- **Unified Catalog**: Discover trending content, categories, and categories from multiple sources.
- **Advanced Video Player**: HLS support, custom controls, and integrated subtitle picker.
- **Performance Optimized**: Route lazy loading, image lazy loading, and intelligent catalog caching.
- **PWA Ready**: Installable on mobile and desktop with offline support.
- **Personalization**: Continue watching history, watchlists, and favorites.
- **Resilient**: Robust proxy handling and automatic addon retries.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Video**: HLS.js
- **Icons**: Lucide React
- **Backend (Proxy)**: Node.js, Express, Axios
- **PWA**: Vite PWA Plugin

## 🚀 Getting Started

### Local Development
1. **Clone & Install**:
   ```bash
   git clone <your-repo-url>
   cd StreamByte
   npm install
   ```
3. Start the application:
   ```bash
   npm run dev
   ```
4. Access the app at `http://localhost:5173`.

## ⚙️ Configuration

- **Addon Integration**: Manifest URLs are managed through the settings page.
- **Debrid Support**: Input your Real-Debrid or Premiumize API keys in Settings for high-speed streaming.
